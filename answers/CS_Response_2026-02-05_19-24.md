### Direct Answer
A database is a systematic collection of data that is organized and stored in a way that allows for efficient retrieval and manipulation. It consists of a collection of data, a schema that defines the structure of the data, and a set of operations that can be performed on the data.

### Verified Facts
* A database management system (DBMS) is software that manages and interacts with the database.
* Databases can be classified into different types, such as relational, NoSQL, graph, and time-series databases.
* Databases are used in a wide range of applications, including web applications, mobile apps, and enterprise software.
